#Android

SlidingMenu 的使用方法

  1. 下载 SlidingMenu 库(可到jfeinstein的github下载也可在我的网盘下载，下方有下载地址)
  2. 将library 导入到工作空间
  3. 引用库, 在你的项目上 alt+Enter 选择Android 右侧内容页下滑到底， add 刚刚倒入的library
  4. 初始化Java代码

接下来即可run

注：如果自己的项目内有v4包的话请自行删除， 因为library库里也有v4包， 所以会导致冲突， 只需把自己的v4包删了，亦或是把library的v4包删了把自己的剪切过去，然后clean 再然后在class内重新导入包即可
