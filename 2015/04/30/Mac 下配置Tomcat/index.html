  2. 设置路径

    一般都是放在 ~/Library/ 目录下，然后重命名一下为 Tomcat，以便查看

  3. 启动 Tomcat

    浏览器转入到 http://localhost:8080/ 就可以看到我们熟悉的小猫了

    同样的是在 bin 目录下

  5. 配置 Tomcat 启动脚本

    将此 code 保存文件名为 tomcat （注意是tomcat），保存到/usr/bin，并赋予文件执行权限
