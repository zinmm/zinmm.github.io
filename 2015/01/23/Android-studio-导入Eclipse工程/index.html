Android studio 导入 Eclipse 工程

1.首先为了避免可能出现的问题把ADT升级到最新版目前是22.6.2

2.右键选择Export并选择Android下的Generate Gradle build fils

导入出完成后并不会导出到其它地方，而是在本地生成了build.gradle文件，在Eclipse可以看到，这个文件是可以被Android Studio识别的

3.随后进入Android Studio 选择Import Project，可以看到刚刚在Eclipse里的项目icon变成一个Android机器人icon，这就说明转换成功，这时候选择工程导入即可

这样就成功的把Eclipse里的工程导入到Android Studio里了！
